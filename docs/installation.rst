============
Installation
============


Stable release
--------------

To install Choeur de la Maison de l'Argentine, run this command in your terminal:

.. code-block:: console

    $ pip install choeur-argentine

This is the preferred method to install Choeur de la Maison de l'Argentine, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Choeur de la Maison de l'Argentine can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/balouf/choeur-argentine

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/balouf/choeur-argentine/tarball/main

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .


.. _Github repo: https://github.com/balouf/choeur-argentine
.. _tarball: https://github.com/balouf/choeur-argentine/tarball/main
